What is Puredom?

puredom is a fast, chainable, exstensible JavaScript library for web applications. In plain english, it makes building developing rich apps using JavaScript, HTML and CSS much easier.

Documentation

For a list of functions you can call on a puredom selection, see the puredom.NodeSelection docs.

For class docs, check out the full puredom documentation. Please note some puredom features are not yet available in the documentation.

Plugins

puredom supports plugins for extending the core selection object.

Check out the available plugins or the plugin documentation.

Download

puredom Light:

  • Puredom Core (selector engine, filters, events, all built-in selection functions)
  • EventEmitter (add eventing to objects and classes)
  • LocalStorage (persistence with adapters)
  • net (http/ajax/jsonp)
  • date (parse & format)
  • Utils:
    • json
    • xml
    • text
    • querystring
    • cookies

Download: puredom.light.zip

puredom Full:

  • All of the light version
  • ControllerManager (work with controllers in an MVC configuration)
  • RouteManager (Manage controllers based on URL patterns)
  • ViewManager (Register, store, load and template HTML, or JSON views)
  • FormHandler (Manage form population, submission and error handling)
  • i18n (Internationalization & localization)
  • NativeAPI (Dynamically create models for APIs using a simple JSON structure)
  • Notifier (Show stylized notifications within the browser window)
  • TestSuite (Run & automate asyncronous tests)

Download: puredom.zip
